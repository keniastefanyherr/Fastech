Sistema avanzado e integrado para el control de inventario, auditoría de procesos y catálogo automatizado de hardware en tiempo real. Diseñado con una interfaz cyberpunk optimizada para la administración de stock y la supervisión de entornos locales, operando bajo una arquitectura desacoplada y eficiente.

---

## 🛠️ Características Principales

* 📊 **Dashboard Central:** Métricas en tiempo real y analíticas rápidas mediante gráficos dinámicos del flujo del sistema.
* 📦 **Control de Inventario:** Módulo CRUD completo e intuitivo para la gestión rigurosa de productos, marcas y unidades disponibles.
* ⚙️ **Panel de Configuración:** Centro de mando interactivo para el control del núcleo, monitoreo de latencia y personalización dinámica de la interfaz.
* 📄 **Auditoría de Procesos:** Consola de logs integrada para supervisar los eventos del sistema en vivo.
